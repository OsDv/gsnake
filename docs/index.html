Gumbo's Snake

By Guembo for gumbos

Controls

Main menu

  • UP/DOWN: Select options (skin,character,color..)
  • RIGHT/LEFT: Chose prefered skin,color...
  • [ENTER]: To start game

Gameplay

  • WASD: For movement
  • P: Pause game
